## Getting Started

This project contains Jenkins Build Methods for Unity.

### Prerequisites

You need to set github credentials before adding package.

### Installation

1. Open UPM
2. Click (+) Sign
3. Click Add package from git URL
4. Past project url
```sh
  https://github.com/moralabs-dev/unity-sdk-build-methods.git
```
5. Install package
